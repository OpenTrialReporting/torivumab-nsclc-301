# RAW-PROVENANCE.md — Simulated Raw Data Development Record
# CTX-NSCLC-301 — Torivumab Phase 3 NSCLC Study

---

## 1. Disclosure

The raw data in `raw/` is **fully synthetic**, generated by simulation scripts in `programs/raw/`.
It is designed to represent what eCRF export data from a clinical data management system (e.g.
Medidata Rave) would look like for a Phase 3 NSCLC immunotherapy study. No real patient records
are represented. All subject IDs, site IDs, dates, and clinical values are simulated.

---

## 2. Design Principle — Treatment Effect Concealment

The simulation embeds a realistic treatment effect (OS HR ≈ 0.62, PFS HR ≈ 0.55, ORR 44% vs 20%)
at the level of random number generation parameters only. No column in any raw CSV encodes
"responder", "benefit", or predicted outcome. The effect is only observable after the full
pipeline: raw → SDTM → ADaM → statistical analysis.

This mirrors real-world practice: raw CRF data contains dates, measurements, and verbatim entries
— not derived endpoints. The clinical story emerges from proper data derivation and analysis.

## 6. Reference Codelists

| File | Purpose | Source |
|---|---|---|
| `raw/codelists/meddra_oncology_subset.csv` | ~80 MedDRA terms (4-level hierarchy) for NSCLC immunotherapy AEs | Publicly documented PT codes from FDA labels, KEYNOTE-024 publications, CDISC examples. **Not a substitute for a licensed MedDRA dictionary.** |
| `raw/codelists/atc_conmed.csv` | ~40 supportive care medications with WHO ATC codes | WHO ATC classification (freely published). |
| `raw/codelists/cdisc_ct.csv` | Key CDISC Controlled Terminology items | CDISC CT (freely downloadable from CDISC website). |

**Licensing note:** Real MedDRA dictionaries require an MSSO licence. WHODrug requires a UMC licence.
For a production study, replace the curated subset with a licensed dictionary and an autocoding engine.
The mapping programs in `programs/sdtm/ae.R` and `programs/sdtm/cm.R` are designed to accept a
full dictionary by replacing the lookup CSV path.

## 7. How to Run

```r
# From project root, in R:
source("programs/raw/00_simulate_raw.R")

# Or from command line:
Rscript programs/raw/00_simulate_raw.R
```

Outputs: 13 CSV files in `raw/`. Reproducible via `set.seed(20260301)` at top of controller.
